# help.provo.edu

# Installation

Requires a .env file in the root directory with the following variables set

```
LDAPHOST=
LDAPPORT=
SQL_ROOT=
HELPMYSQL_USER=
HELPMYSQL_PASSWORD=
HELPMYSQL_DATABASE=
HELPMYSQL_HOST=
HELPMYSQL_PORT=
VAULT_READ_HOST=
VAULT_READ_USER=
VAULT_READ_PASSWORD=
VAULT_READ_DATABASE=
ROOTDOMAIN=
UDPGRAYLOGHOST=
GRAYLOGIP=
GRAYLOGPORT=
SWHELPDESKHOST=
SWHELPDESKUSER=
SWHELPDESKPASSWORD=
SWHELPDESKDATABASE=
```
